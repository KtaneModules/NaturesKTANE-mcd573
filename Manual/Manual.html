Keep Talking and Nobody Explodes Mod Natures

On the Subject of Natures

Sounds can be heard coming from inside! This egg will hatch soon!

To disable this module, you must find the bomb's creature type, and set it's corresponding competitive nature by re-breeding.

Finding the Target Nature

Use the table below to match the child creature with the last digit of the serial number in order to find it's competitive nature.

Traplot Runtin Alot Peren Morkie
0-1 Adamant Afraid Stubborn Shy Neutral
2-3 Shy Stubborn Neutral Afraid Adamant
4-5 Neutral Afraid Adamant Shy Stubborn
6-7 Stubborn Neutral Afraid Adamant Shy
8-9 Afraid Adamant Shy Stubborn Neutral
  • If the last digit of the serial number is zero or one, then this is your target nature. Re-breed for it and battle.
  • Otherwise, adjust the nature based on the list below.

Adjusting the Nature

When using this list, a +[Number] or -[Number] means you need to re-calculate your target nature by adding/subtracting the number to the last digit of the serial number.

After recalculating, there is no need to go through this list again. Simply Re-Breed for the new calculated nature and select the Battle! button.

If the nature value goes negative after calculating, make it zero. If the nature value goes over 10 after calculating, use the least significant digit.

Follow the first rule that applies:

  1. If there is another Natures module on this bomb, do not adjust the target nature.
  2. Otherwise, if there is a "Monsplode Trading Cards" or "Monsplode, Fight!" module on the bomb, the target nature is modified by -3.
  3. Otherwise, if there is a boss module on the bomb, such as Forget Me Not or Souvenir, the target nature is modified by +2.
  4. Otherwise, if there is more than 2 batteries on the bomb, the target nature is modified by +1.
  5. Otherwise, if the child creature is an Alot, the target nature is modified by -2.
  6. Otherwise, if the bomb's serial number contains a vowel, the target nature is modified by -1.
  7. Otherwise, if one of the parent creatures is a Trapalot, the target nature is modified by +5.
  8. Otherwise, if one of the parent creatures has an Adamant nature, the target nature is Adamant.
  9. Otherwise, if one of the parent creatures has a Shy nature, the target nature is set to 0.
  10. Otherwise, if one of the parent creatures has an Afraid nature, the target nature is subtracted by the number of port plates.
  11. Otherwise, if there is a port plate with a PS2 port, this creature is hacked. There is no need to adjust the target nature.

Strikes

The following situations will warrant a strike from the module:

  • Re-breeding on the correct nature.
  • Battling with an incorrect nature.
